About Cleanify

HTML & CSS framework made from scratch based on One UI (Samsung) and Bootstrap.

Emerged from the need to mix both styles in a only framework for web design.

Dark Mode

Cleanify implement a dark variant of his design.

You can choose which one to use, or you can simply use both.
